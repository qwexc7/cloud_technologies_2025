## Немного теории

IaaS (Infrastructure as a Service) — это облачная модель, где провайдер предоставляет базовые IT-ресурсы (виртуальные серверы, хранилища, сети)

PaaS (Platform as a Service) — это модель облачных вычислений, которая предоставляет разработчикам готовую облачную платформу с операционными системами, базами данных, серверами, инструментами разработки и средами выполнения

SaaS (Software as a Service) — это модель доставки ПО через интернет, где поставщик размещает приложение на своих серверах, а пользователи получают к нему доступ через браузер или приложение без необходимости установки и обслуживания локально

## Практическая часть 

IT Tower - верхний уровень иерархии, "Какой класс ИТ-функций это обслуживает??"<br>
Service Family - группировка по функциональному назначению, "Для какой задачи бизнес использует этот сервис?"<br>
Service Type - название конкретного продукта от провайдера<br>
Service Sub Type - "За что именно взимается плата внутри этого сервиса?"<br>
Service Usage Type - уточняет конкретную метрику<br>
